Detailed Description

GINPUT allows it to interface toggle buttons easily to your application.

Precondition
GFX_USE_GINPUT must be set to TRUE in your gfxconf.h
GINPUT_NEED_TOGGLE must be set to TRUE in your gfxconf.h
