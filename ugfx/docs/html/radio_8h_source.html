58 /**
59  * @brief Create a radio widget.
60  * @return NULL if there is no resultant drawing area, otherwise a window handle.
61  *
62  * @param[in] g The GDisplay to display this window on
63  * @param[in] gb The GRadioObject structure to initialise. If this is NULL the structure is dynamically allocated.
64  * @param[in] pInit The initialisation parameters
65  * @param[in] group The group of radio buttons this radio button belongs to.
66  *
67  * @note Only one radio button in any group is ever pressed at one time. Pressing one radio button will
68  * release all others in the group.
69  * @note The drawing color and the background color get set to the current defaults. If you haven't called
70  * @p gwinSetDefaultColor() or @p gwinSetDefaultBgColor() then these are White and Black respectively.
71  * @note The font gets set to the current default font. If you haven't called @p gwinSetDefaultFont() then there
72  * is no default font and text drawing operations will no nothing.
73  * @note A radio button remembers its normal drawing state. If there is a window manager then it is automatically
74  * redrawn if the window is moved or its visibility state is changed.
75  * @note A radio button supports mouse and a toggle input.
76  * @note When assigning a toggle, only one toggle is supported. If you try to assign more than one toggle it will
77  * forget the previous toggle. When assigning a toggle the role parameter must be 0.
78  *
79  * @api
80  */
