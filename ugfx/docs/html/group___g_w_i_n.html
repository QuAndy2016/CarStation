Console Windows need BaseStreamSequential support (ChibiOS only)

Defaults to FALSE

Note
To use the ChibiOS basestream functions such as chprintf() for printing in a console window you need to set this option to TRUE in your gfxconf.h and include in your application source file... #include "chprintf.h" In your makefile, as part of your list of C source files, include ${CHIBIOS}/os/various/chprintf.c
