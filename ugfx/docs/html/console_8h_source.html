47 /**
48  * @brief Create a console window.
49  * @details A console window allows text to be written.
50  * @note Text in a console window supports newlines and will wrap text as required.
51  * @return NULL if there is no resultant drawing area, otherwise a window handle.
52  *
53  * @param[in] g The GDisplay to display this window on
54  * @param[in] gc The GConsoleObject structure to initialise. If this is NULL the structure is dynamically allocated.
55  * @param[in] pInit The initialization parameters to use
56  *
57  * @note The drawing color and the background color get set to the current defaults. If you haven't called
58  * @p gwinSetDefaultColor() or @p gwinSetDefaultBgColor() then these are White and Black respectively.
59  * @note The font gets set to the current default font. If you haven't called @p gwinSetDefaultFont() then there
60  * is no default font and text drawing operations will no nothing.
61  * @note On creation even if the window is visible it is not automatically cleared.
62  * You may do that by calling @p gwinClear() (possibly after changing your background color)
63  * @note A console does not save the drawing state. It is not automatically redrawn if the window is moved or
64  * its visibility state is changed.
65  *
66  * @api
67  */

119 /**
120  * @brief Print a formatted string at the cursor position in the window. It will wrap lines as required.
121  * @details This function implements a minimal printf() like functionality
122  * The general parameters format is: %[-][width|*][.precision|*][l|L]p.
123  * The following parameter types (p) are supported:
124  * - <b>x</b> hexadecimal integer.
125  * - <b>X</b> hexadecimal long.
126  * - <b>o</b> octal integer.
127  * - <b>O</b> octal long.
128  * - <b>d</b> decimal signed integer.
129  * - <b>D</b> decimal signed long.
130  * - <b>u</b> decimal unsigned integer.
131  * - <b>U</b> decimal unsigned long.
132  * - <b>c</b> character.
133  * - <b>s</b> string.
134  * @note Uses the current foreground color to draw the string and fills the background using the background drawing color
135  *
136  * @param[in] gh The window handle (must be a console window)
137  * @param[in] fmt The format string (as per printf)
138  * @param[in] ... The format string arguments.

140  * @api
